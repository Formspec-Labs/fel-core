# Code of Conduct

This project expects respectful, direct, technical collaboration.

## Expected Behavior

- Discuss ideas, code, and evidence rather than personal traits.
- Be clear about tradeoffs, uncertainty, and impact.
- Respect maintainer decisions about scope, timing, and release readiness.
- Report security issues through the private process in [SECURITY.md](SECURITY.md).

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private security reports before coordinated disclosure.
- Repeated off-topic argument after a maintainer has closed the discussion.
- Disruptive behavior that prevents maintainers or contributors from doing the
  work of the project.

## Enforcement

Maintainers may edit, hide, or delete comments; close issues; block accounts;
or decline contributions that violate this code. Serious incidents should be
reported privately to the maintainers.
